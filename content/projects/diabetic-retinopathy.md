## Overview

An **AI-powered system** for real-time analysis of fundus images to detect diabetic retinopathy at various severity levels using advanced deep learning techniques.

---

## Key Features

- 🧠 **Ensemble CNN models** with CBAM (Convolutional Block Attention Module) attention mechanism
- 🔬 **Binary and multi-class classification** of DR severity levels
- ⚡ **Real-time image processing** and analysis pipeline
- 🌐 **Django web application** for accessible, real-time prediction
- 📊 Detailed **severity classification and reporting**
