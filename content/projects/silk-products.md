## Overview

A comprehensive **Django-based** web application for managing silk product inventory, featuring role-based access control, REST API integration, and a responsive user interface.

---

## Key Features

- ✅ **Role-based access control** system for multi-user management
- ✅ **REST APIs** built with Django REST Framework
- ✅ **Authentication system** with JWT for secure access
- ✅ **Advanced search** and filtering capabilities
- ✅ **Email notification** system for inventory alerts
- ✅ **Responsive design** across all device sizes
- ✅ Complete **CRUD operations** for inventory management
